##  **Project Overview**

**Student Attendance System** allows teachers or administrators to track students’ attendance day-by-day, generate reports, sort results, search by name, and save reports to a file.

This is an educational project demonstrating:
- Working with arrays and matrices
- File operations (`System.IO`)
- Sorting algorithms
- Console interface development

---

##  **Features**

| Feature | Description |
|--------|------------|
| ✔ Mark attendance | Mark attendance for a specific day (Present/Absent) |
| ✔ Attendance table | Display a full attendance table |
| ✔ Report generator | Percentage-based attendance report |
| ✔ Save report | Save results to `attendance_report.txt` |
| ✔ Load report | Load previously saved report |
| ✔ Search | Search student by name |
| ✔ Sort | Sort students by attendance percentage |
| ✔ User-friendly console UI | Interactive menu with instructions |
